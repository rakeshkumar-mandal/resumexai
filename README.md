# ResumeXAI

AI-powered MERN stack resume builder with smart suggestions and modern UI.

## Features

- AI Resume Generation
- User Authentication
- Resume Templates
- PDF Export
- Responsive Design
- MERN Stack Architecture

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB

## Folder Structure

```bash
client/   -> Frontend
server/   -> Backend
```

## Installation

### Clone Repository

```bash
git clone https://github.com/rakeshkumar-mandal/resumexai.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```

## Environment Variables

Create `.env` file inside server folder.

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```
